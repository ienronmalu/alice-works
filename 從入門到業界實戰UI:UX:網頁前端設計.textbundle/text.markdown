## 設計原則
* 一致性
* 簡約
* 資訊清晰
* 維持用戶優越感

### 一致性
讓使用者合乎預期結果
	* 外觀上
	* 功能上
		雲端同步/行動裝置同步
		微互動：下拉、轉場動畫特效
	* 介面圖示
		編輯？新增文章？
![](assets/assets/0425C21F-4305-4B52-9648-D98A52870719.png)
		內部視覺一致	官方內部平台
		外部視覺一致	很多產品線的平台開發/維護
						每天討論代表公司文化的標準規範，使用戶養成習慣這樣的體驗流程(Apple, Adobe)
![](assets/assets/28C0231B-E67E-4F0E-8401-5F44C8B6A503.png)
